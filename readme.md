# How to Use

To ad a task:
```
click [new task] or CTRL + N
enter details
click save
```

To edit a task:
```
double click on the task
edit details
click save
```

To complete/incomplete a task:
```
edit a task
click complete/incomplete
```

To delete a task:
```
edit a task
click delete
```

To edit settings
```
click settings
edit settings
click save
```
